Exemplo 10 HTML - Formulários

Os formulários em HTML são um dos principais recurso para coleta de dados, eles devem ser simples e altmante funcionais, atendendo sempre as necessidades dos usuários.

Para trabalhar com fomulários em html, possuimos uma série de componentes que vão facilitar o processo de interação com o usuário. Todos os formulários começam necessariamente com a tag <form>, essa tag indica para o navegador que página html possui um formulários e que pode enviar dados.

Além da tag <form> temos também que usar os atributos action (que define o local - url- em que os dados recolhidos/coletados devem ser enviados.), e o method que define qual o método http para enviar os dados GET ou POST
