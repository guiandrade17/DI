Web Semântica

A web semântica é um conceito que visa organizar os dados na web, permitindo que máquinas os compreendam e enterajam com ela (WEB).Para isso usamos várias tags semânticas que facilitam o trabalho dos mecanismos de busca.

Neste exemplo exemplo estamos usando a web semântica na prática, use um bloco de notas para acessar o documento html e ver como essa página foi construída.

React

React é uma biblioteca JavaScript para construir interfaces de utilizador (UI) e aplicações web de forma eficiente, usando componentes reutilizáveis e um Virtual DOM para atualizações rápidas da página. Permite a criação de interfaces complexas a partir de peças menores e independentes, tornando o desenvolvimento mais rápido e organizado, com benefícios para a experiência do utilizador e SEO.

Angular

Angular é um framework de código aberto, mantido pelo Google, para desenvolver aplicações web e móveis robustas e interativas. Ele utiliza TypeScript como linguagem principal, seguindo uma arquitetura baseada em componentes, o que permite reutilizar código e organizar projetos de forma eficiente. Suas principais características incluem uma CLI (Command Line Interface) para automatizar tarefas, um conjunto completo de ferramentas para diversas funcionalidades (como roteamento e gerenciamento de estado) e uma grande comunidade de desenvolvedores.
