Exemplo 6 HTML - Tags de Seção/Títulos

Este exemplo mostra como usar tags de título(<h1> a <h6>) para organizar o conteúdo em seções hierárquicas.

Descrição

As tags de título são usadas para definir cabeçalhos e subtítulos. Elas variam de <h1> (mais importante) ao <h6> menos importante.
